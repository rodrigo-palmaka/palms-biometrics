# Privacy Policy

**Last updated: June 2026**

## What this is

Palms-Biometrics is a personal, single-user health data aggregation tool. It is not a commercial product or public service.

## Data collection

This application accesses health data (sleep, heart rate, HRV, activity) from connected services — currently Oura Ring — on behalf of the sole user (the account owner who authorized the application).

## Data storage

All data is stored locally on the user's own machine in a SQLite database file. No data is transmitted to any third-party server, cloud service, or external storage. No data is ever shared with anyone.

## Third-party services

This application connects to the Oura API solely to retrieve the authenticated user's own health data, using OAuth2 tokens granted by that user. No other parties have access to that data.

## Contact

This is a private personal project. For questions, contact the repository owner directly via GitHub.
